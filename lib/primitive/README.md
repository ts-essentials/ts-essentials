`Primitive` matches any [primitive value](https://developer.mozilla.org/en-US/docs/Glossary/Primitive)

```ts
let primitive: Primitive;

primitive = 1;
primitive = "1";
primitive = true;
primitive = 1n;
primitive = Symbol.iterator;
primitive = null;
primitive = undefined;
```
